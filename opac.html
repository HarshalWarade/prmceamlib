OPAC

PRMCEAM Library is automated using LIBMAN software

Online OPAC (Online Public Access Catalogue)

Libman software provides the facility of using OPAC, through which they can gather details on current library collection and book status. User can search Books using different tags like: Title, Author, Publisher, Subject and Keywords.

User would be able to see information regarding the book like Author, Publishers, Publish year, Accession numbers with Series code, ISBN, Call No (Classification Number + Book Number), Quantity of Book.


MOPAC(Mobile Online Access Catalogue)

In the current scenario Libman provides remote acces facility like MOPAC which helps the users to get benefits of OPAC by using android mobile like to search details on current library collection and book status.


Library Visitors Management

By scanning the QR code displayed on the entrance of the Library Reading Room and Stack Area users can check in and check out in to Library Reading Room and mark his/her attendance in the library to get benefits of Library Reading Room or for accessing other resources from Library stack Area.
